# TrendNest

TrendNest is a frontend-only static website prototype for GitHub Pages.

## What it is

A premium one-page investor-demo website built with:
- `index.html`
- `style.css`
- `script.js`

No backend, no framework, no build step, and no special hosting are required.

## Deploy to GitHub Pages

1. Upload `index.html`, `style.css`, `script.js`, and any optional assets to your GitHub repository.
2. Enable GitHub Pages for the repository.
3. The deployed site will open the real TrendNest website directly from `index.html`.

## Notes

- This is a mock/demo-only frontend prototype.
- All interactions are powered by vanilla JavaScript.
- All data is local and static.
